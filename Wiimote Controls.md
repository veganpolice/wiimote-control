# Wiimote Controls Cheat Sheet

## Default Mode (no buttons held)

| Button | Action |
|--------|--------|
| **A** | Left click (tap) / Drag (hold + move stick) / Double-click (double-tap) |
| **B** | Toggle Wispr Flow (voice input) |
| D-pad Left | Arrow Left |
| D-pad Right | Arrow Right |
| D-pad Up | Arrow Up |
| D-pad Down | Arrow Down |
| **+** | Enter / Return |
| **-** | Tab |
| **Home** | Mission Control (Ctrl+Up) |
| **1** | Escape |
| **2** | Spotlight / Siri (Cmd+Space) |

## Nunchuk

| Input | Action |
|-------|--------|
| **Stick** | Move cursor |
| **Z tap** | Enter / Return |
| **Z hold** | Precision cursor + Z-mode (see below) |
| **C hold** | C-mode modifier (see below) |
| **C + Stick** | Scroll (up/down/left/right) |

## Hold Z — Text Editing Mode

| Button | Action |
|--------|--------|
| A | Copy (Cmd+C) |
| B | Cut (Cmd+X) |
| + | Paste (Cmd+V) |
| - | Undo (Cmd+Z) |
| D-pad Left | Select char left (Shift+Left) |
| D-pad Right | Select char right (Shift+Right) |
| D-pad Up | Select line up (Shift+Up) |
| D-pad Down | Select line down (Shift+Down) |

## Hold C — Quick Actions Mode

| Button | Action |
|--------|--------|
| D-pad Left | Previous app (Cmd+Shift+Tab) |
| D-pad Right | Next app (Cmd+Tab) |
| D-pad Up | Scroll to top (Cmd+Up) |
| D-pad Down | Scroll to bottom (Cmd+Down) |
| + | Redo (Cmd+Shift+Z) |
| - | Undo (Cmd+Z) |
| A | New tab (Cmd+T) |
| Home | Switch window (Cmd+`) |
| 1 | Interrupt (Ctrl+C) |
| 2 | Close tab (Cmd+W) |

## Hold B — Workspace & System Mode

| Button | Action |
|--------|--------|
| D-pad Left | Workspace 1 (Cmd+1) |
| D-pad Right | Workspace 2 (Cmd+2) |
| D-pad Up | Workspace 3 (Cmd+3) |
| D-pad Down | App windows (Ctrl+Down) |
| + | Zoom in (Cmd+=) |
| - | Zoom out (Cmd+-) |
| Home | Lock screen (Ctrl+Cmd+Q) |
| A | Select All + Copy (Cmd+A, Cmd+C) |
| Nunchuk Z | Paste (Cmd+V) |
| Nunchuk C | Copy (Cmd+C) |

*B tap (no combo) = Copy (Cmd+C)*

## Hold Home — Tabs & Pages Mode

| Button | Action |
|--------|--------|
| D-pad Left | Previous tab (Cmd+Shift+[) |
| D-pad Right | Next tab (Cmd+Shift+]) |
| D-pad Up | Page Up |
| D-pad Down | Page Down |
| A | Quick app switch (Cmd+Tab) |
| + | Zoom in (Cmd+Shift+=) |
| - | Zoom out (Cmd+-) |
| 1 | Interrupt (Ctrl+C) |
| 2 | Close tab (Cmd+W) |
| Nunchuk Z | Undo (Cmd+Z) |
| Nunchuk C | Toggle sticky scroll |

*Home tap (no combo) = Spotlight (Cmd+Space)*

## Gestures (Accelerometer)

| Gesture | Action |
|---------|--------|
| Flick left | Previous app (Cmd+Shift+Tab) |
| Flick right | Next app (Cmd+Tab) |
| Flick up | Volume up |
| Flick down | Volume down |
| Shake | Undo (Cmd+Z) |

## Cursor Settings

- **Dead zone:** 0.1 (stick values within this radius ignored)
- **Max speed:** 160 px/tick at full deflection
- **Curve:** Quadratic (finer control near center)
- **Z held:** Precision mode (slower cursor)
- **C held:** Stick becomes scroll wheel
- **Scroll speed:** 60x multiplier
